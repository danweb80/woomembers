WooMembers

O objetivo deste plugin para WordPress/WooCommerce é fazer a integração das vendas do WooCommerce com a plataforma The Members.
(Baseado no plugin llregonproduct v2.2 usado para fazer a mesma função com a plataforma de Ead da LeadLovers.)

Author: Daniel Weber (prof.daniel.weber@gmail.com)
License: GPLv2 or later

v1.0
    Features:
    - Cria uma aba 'The Members' nas configurações do WooCommerce para configurações com a a API da plataforma;
        - IMPORTANTE: É necessário criar um usuário desenvolvedor junto à plataforma conforme orientação fornecida na Documentação da API (https://documentation.themembers.dev.br/api/)

    - Cria uma aba 'The Members' nas configurações dos Produtos do WooCommerce (Panels) para habilitar a conexão do status de peddo do produto com um produto da plataforma
